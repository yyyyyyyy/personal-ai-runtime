# Positioning

## 我们是谁

一个**让你的 AI 换模型不丢记忆**的开源产品。

不是模型公司。不是 Chat UI。不是 Agent 框架。

是 AI 与你之间的**关系层**：承载你的记忆、偏好、目标和决策——独立于任何模型或厂商。

---

## 核心差异化

### 跨模型连续性

换模型只需改配置。记忆完全保留。这是架构级的保证，不是功能。

### 数据主权

你的数据永远在你自己的机器上。完整可导出。数据永不出本机。

### 治理机制

高风险操作（写文件、发邮件、执行命令）必须经过你审批。不是依赖模型对齐，而是架构级约束。

---

## 我们的诚实自评

### 我们做得好的

- **跨模型连续性**：换模型不丢记忆。这是架构级的保证，不是功能。
- **数据主权**：你的数据永远在你自己的机器上。完整可导出。
- **治理机制**：高风险操作必须审批。不是依赖模型对齐，而是架构级约束。
- **自我修复**：后台任务失败后自动诊断和重新规划（当前限于 Background Worker 任务）。

### 我们做得不够好的

- **开箱体验**：当前需要自己配置 `.env` 和 LLM API Key。没有内置默认模型。
- **移动端**：Web UI 在手机上体验未优化。没有原生 App。
- **多模态**：图片理解尚未支持。
- **用户量**：早期项目。社区和生态尚小。

### 谁应该用我们

- 每天与 AI 高频互动、积累了大量偏好和记忆的用户
- 关注数据主权和隐私的用户
- 在不同场景使用不同模型的用户
- 愿意自己部署和配置的技术用户

### 谁可能不适合

- 需要一键开箱即用的用户（当前）
- 需要移动端原生体验的用户（当前）
- 不需要长期记忆延续性的用户
